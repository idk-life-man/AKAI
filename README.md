\# AKAI - Personal AI Assistant



A self-hosted AI assistant built from scratch on Windows.



\## Features

\- Multi-model support: DeepSeek V3, DeepSeek R1, Ollama (local)

\- Persistent memory across sessions

\- Automatic web search via Tavily

\- RAG knowledge base with PDF/text ingestion

\- Agent mode: file read/write, Python execution



\## Stack

\- Python, Streamlit, ChromaDB, SentenceTransformers

\- DeepSeek API, Ollama, Tavily



\## Setup

1\. Clone the repo

2\. Create a virtual environment and install dependencies

3\. Add API keys to `config/.env`

4\. Run `streamlit run projects/app.py`
